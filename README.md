# AI Invoice & Expense Analyzer for MSMEs

This application allows small businesses to upload invoice images and automatically extract structured expense data using AI.

## Features
- Invoice OCR
- AI-based data extraction
- Expense categorization
- Excel download
- Simple web interface

## Usage
1. Upload invoice image
2. View extracted data
3. Download Excel

## Built With
- Python
- Streamlit
- Gemini AI
